# 🍔 Food Memory Match Game

A fun and interactive memory card matching game built using HTML, CSS, and JavaScript. Players flip food-themed cards to find matching pairs while tracking moves, time, and best score.

## 🚀 Features

- 🍕 Food-themed memory cards
- 🎮 Easy, Medium, and Hard difficulty levels
- ⏱️ Timer
- 🔢 Move counter
- 🏆 Best score tracking using Local Storage
- 🔄 Restart game option
- 🎉 Win popup with game statistics
- 🎨 Responsive and attractive UI
- ✨ Smooth card flip animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone this repository.
2.   Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click on index.html.
5. Select *Open with Live Server*.
6. Play the game in your browser.
